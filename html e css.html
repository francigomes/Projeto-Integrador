HTML e CSS

HTML e CSS são duas das linguagens fundamentais para a construção de páginas na web. Embora trabalhem juntas, cada uma possui um objetivo diferente e complementam-se para criar sites funcionais e visualmente agradáveis.

HTML – Linguagem de Estrutura

HTML (HyperText Markup Language) é uma linguagem de marcação usada para estruturar o conteúdo.

Exemplo da Estrutura HTML

• (h1) cria um título
• (p) cria um parágrafo
• (img) exibe uma imagem
• (a) cria um link clicável

CSS – Linguagem de Estilo

CSS controla cores, fontes, tamanhos, espaçamento, animações e layouts.

Exemplo da Estrutura CSS

• (font-size) controla o tamanho
• (width) define a largura
• (border-radius) arredonda bordas
• (text-align) centraliza texto
